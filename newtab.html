Background image search tags (Optional):

Separate search terms by commas with no spaces. Images are randomly sourced from source.unsplash.com using these search tags. Example: forest,ocean,city

Time to wait before getting new image (in minutes) (Optional):

Evergreen won't download a new background if a new one has been downloaded this many minutes ago. Saves data. Set to 0 to disable.
